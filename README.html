FizzBuzz with Sitecore 8.1 Update 3 MVC

Rules are super simple, count number one after another but when the number:

  • is dividable by 3 say Fizz
  • is dividable by 5 say Buzz
  • is dividable by 3 and 5 say Fizz Buzz
  • otherwise the number The expected outptut for the number 16 would be, 1,2,fizz,4,buzz,fizz,7,8,fizz,buzz,11,fizz,13,14,fizzbuzz,16....
    • otherwise the number.

This is a demo of combining ASP.NET MVC + Sitecore 8.1 Update 3 with MVC + NUnit + Unity dependency injection + AngularJS + Twitter Bootstrap
